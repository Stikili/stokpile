## Getting Started

### Creating an Account

1. Visit the Stokpile app
2. Click "Sign Up"
3. Enter your email and password
4. Verify your email (if required)
5. Complete your profile

### Your First Group

**Option 1: Create a New Group**
1. Click "Create Group"
2. Enter group name and description
3. Set contribution frequency
4. Enable/disable payouts
5. Click "Create"

**Option 2: Join an Existing Group**
1. Get an invite link from the group admin
2. Click the link
3. Accept the invitation
4. You're now a member!

## Managing Groups

### Switching Between Groups

Use the group selector at the top of the page to switch between your groups.

### Group Roles

- **Admin** - Full control (create/edit/delete)
- **Member** - View and add contributions

### Inviting Members

**As an Admin:**
1. Go to Group Info tab
2. Click "Invite Members"
3. Share the invite link or code

### Managing Join Requests

Admins can approve or deny join requests in the Dashboard.

## Contributions

### Adding a Contribution

1. Go to Contributions tab
2. Click "Add Contribution"
3. Enter amount and date
4. Add optional note
5. Click "Save"

### Viewing Contributions

- See all contributions in the Contributions tab
- Filter by member or date
- View totals and statistics

### Editing/Deleting Contributions

**Admins can:**
- Edit any contribution
- Delete contributions
- View full contribution history

**Members can:**
- View all contributions
- Add their own contributions

## Payouts

> Only available if enabled for the group

### Scheduling a Payout

**Admins only:**
1. Go to Payouts tab
2. Click "Schedule Payout"
3. Select recipient
4. Enter amount and date
5. Add optional note
6. Click "Schedule"

### Viewing Payouts

- See scheduled and completed payouts
- Filter by recipient or status
- View payout history

---

## Meetings

### Scheduling a Meeting

**Admins only:**
1. Go to Meetings tab
2. Click "Schedule Meeting"
3. Enter title, date, and location
4. Add agenda (optional)
5. Click "Schedule"

### Meeting Features

Each meeting has:
- **Details** - Date, time, location, agenda
- **Voting** - Create polls for member votes
- **Notes** - Meeting minutes and notes
- **Chat** - Real-time discussion

### Attendance

Admins can mark attendance for each meeting.

## Members

### Viewing Members

Go to Group Info tab to see all members and their roles.

### Member Stats

Click on any member to see:
- Total contributions
- Recent activity
- Participation rate

### Removing Members

**Admins can:**
1. Go to Group Info
2. Click on member
3. Select "Remove from Group"
4. Confirm removal

## Settings

### Profile Settings

1. Click your avatar (top right)
2. Select "Profile"
3. Update name, email, or password
4. Save changes

### Group Settings

**Admins can:**
- Edit group name
- Edit description
- Change contribution frequency
- Enable/disable payouts
- Upload constitution
- Delete group

### Theme

Toggle between light and dark mode in the header.

## FAQ

### How do I recover my password?

Click "Forgot Password" on the sign-in page and follow the instructions.

### Can I be in multiple groups?

Yes! You can create or join unlimited groups.

### Who can see my contributions?

All members of the group can see all contributions.

### Can I leave a group?

Yes, click your profile menu and select "Leave Group".

### What happens if I delete a group?

**Admins:** All data is permanently deleted. This cannot be undone.

### Is my data secure?

Yes, all data is encrypted and protected with row-level security.
